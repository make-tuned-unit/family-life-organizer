# FamilyLife iOS Roadmap

Last updated: 2026-04-07

## Now

- Stabilize the navigation structure and reduce root-tab overload.
- Surface network failures clearly instead of silently failing.
- Stop aggressive permission prompting.
- Establish a single production-readiness checklist in-repo.

## Next

- Move `Decisions`, `Rivalries`, and `Gifts` from local-only persistence to shared backend data.
- Ship weekly meal planning with grocery generation and pantry links.
- Add EventKit calendar sync and availability overlays.
- Add widgets, App Intents, and Spotlight indexing for the highest-frequency workflows.

## Later

- Add automations and family digest workflows.
- Expand memory capture into a household knowledge layer.
- Add shared expense splitting and settlement tracking.
- Introduce proactive planning and recommendation features once the shared data model is stable.
